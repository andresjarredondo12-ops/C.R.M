# AME Legal CRM

CRM completo para estudios jurídicos desarrollado en un solo archivo HTML + CSS + JavaScript.

**Características:**
- Sistema de login / registro / recuperación de contraseña
- Usuario demo automático: `demo@ame-legal.com` / `123456`
- Gestión completa de Clientes (crear, buscar, eliminar)
- Expedientes y Casos (vinculados a clientes con estados)
- Agenda de tareas y citas (con vencimientos y estado)
- Todo guardado en el navegador con `localStorage`
- Diseño moderno y responsive

### Cómo usarlo

1. Descarga el archivo `index.html`
2. Ábrelo directamente en cualquier navegador (Chrome, Edge, Firefox)
3. Inicia sesión con las credenciales de demo
